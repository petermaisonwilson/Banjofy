===========================================================

Status
------
READY FOR WINDOWS BUILD AND ACCEPTANCE TEST

Confirmed baseline
------------------
Module 8B Build 001.

Added
-----
- Genuine BPM detection from downloaded audio.
- Bundled FFmpeg helper; no separate FFmpeg installation required.
- Supports YouTube WebM/M4A and other FFmpeg-readable audio.
- Analyses up to the first 180 seconds.
- Stores detected BPM in Analysis and Library records.
- Recalculates estimated bars using the detected BPM.
- Practice beat-grid timing uses the detected BPM.
- Safe fallback to 92 BPM when audio cannot be decoded or no reliable beat is found.
- Analysis note records whether BPM was detected or fallback was used.

- No real chord detection yet.
- Provisional G/C/G/D chord labels remain only for grid plumbing.
- BPM can still be affected by unusual rhythm, rubato, spoken intros, or half/double-time ambiguity.

- Beat-grid cursor speed changes appropriately between slow and fast songs.
- Undecodable or rhythmically unclear audio falls back safely rather than crashing.
