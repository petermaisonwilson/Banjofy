# Banjofy

## Build 003 - YouTube Search Interface

Build 003 starts the real music-import process without adding audio download yet.

What works in this build:

- On-screen version says **Banjofy 0.3.0 - Build 003 YouTube Search**.
- Existing demo player/timing-grid features remain in place.
- YouTube search is now active.
- Results display title, channel/uploader, duration and a thumbnail where available.
- Clicking **Use for Demo** selects that YouTube result and keeps the existing demo timing grid for now.
- Status messages make clear that audio download/playback is coming next.

What is not included yet:

- Real YouTube audio download.
- Real sound playback.
- Automatic BPM/chord analysis.

Those are planned for following builds once the search/selection workflow is tested.
